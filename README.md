## 📦 Project Overview
This repository provisions a basic VPC on AWS using Terraform.

**Stack:**
- Terraform
- AWS Provider

## 🚀 How to Use
```bash
export AWS_ACCESS_KEY_ID=your_access_key
export AWS_SECRET_ACCESS_KEY=your_secret_key
terraform init
terraform plan
terraform apply
```

## 🔧 Customization
Edit `variables.tf` or use a `terraform.tfvars` file.
